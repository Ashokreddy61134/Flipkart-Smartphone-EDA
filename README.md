This project is about analyzing smartphone data collected from Flipkart. The data includes details like mobile name, price, rating, brand, and features.

First, the data is collected using web scraping. Then, it is cleaned and prepared for analysis. After that, Exploratory Data Analysis (EDA) is done using Python tools like Pandas, Matplotlib, and Seaborn.

The main goal is to understand mobile prices, popular brands, and customer ratings. This helps to get useful insights about the smartphone market.
